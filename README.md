# todo-card
A clean and responsive Todo Card built with HTML, CSS, and JavaScript for the HNG Stage 0 Frontend task. It displays a task with details such as title, description, priority, status, due date, and time remaining. It also includes interactive features like a checkbox to mark completion and/delete buttons.
This was updated by extending the stage 0 version by introducing state management, edit functionality, dynamic status control, expandable content, and improved time handling.
